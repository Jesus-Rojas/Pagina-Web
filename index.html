Sistemas Operativos

Mac

macOS es una serie de sistemas operativos gráficos desarrollados y comercializados por Apple Inc. desde 2001. Es el sistema operativo principal para la familia de computadoras Mac de Apple.

Windows

Windows es el nombre de una familia de distribuciones de software para PC, teléfonos inteligentes, servidores y sistemas empotrados, desarrollados y vendidos por Microsoft y disponibles para múltiples arquitecturas, tales como x86, x86-64 y ARM.

Linux

Linux es una familia de sistemas operativos de código abierto tipo Unix basados ​​en el kernel de Linux, un kernel del sistema operativo lanzado por primera vez el 17 de septiembre de 1991 por Linus Torvalds.
